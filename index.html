Email Reminders
Important: The Owner field in each order must be the person's Gmail address (e.g. rajesh@gmail.com).

Two reminder types are available:
⚠ Daily Alerts — urgent only: overdue deadline, due today/tomorrow, pending approvals. Runs every morning.
📋 3-Day Summary — full personal summary of all orders, sent every 3 days. Includes running, not started, upcoming dispatches, recently completed.
Trigger Setup (one-time in Apps Script):
Apps Script → clock icon (Triggers) → + Add Trigger
① Function: sendDailyReminders → Time-driven → Day timer → 8am–9am
② Function: sendEvery3DaySummary → Time-driven → Day timer → 8am–9am
